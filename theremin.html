An online Theremin instrument

The theremin is an electronic musical instrument invented by Leon Theremin in 1920s.

This demo uses pizzicato.js to implement a "virtual" Theremin on your browser.

The pizzicato.js library (and other similar ones) use the Web Audio API to play sounds on the browser.
